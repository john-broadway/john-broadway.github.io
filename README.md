Source for [john-broadway.github.io](https://john-broadway.github.io).

## Editing

Open `index.html`. All styles are in the `<style>` block at the top. Color tokens live in `:root`. Sections are commented:

```
/* ── Header ─────────── */
/* ── Section headings ── */
/* ── Cards ──────────── */
/* ── Tags ───────────── */
/* ── Stats grid ─────── */
/* ── Values list ─────── */
/* ── Elsewhere row ───── */
/* ── Buddy block ─────── */
/* ── Footer ─────────── */
```
